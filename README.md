# Monster Money Manager

Monster Money Manager is a mobile budgeting app that turns personal finance into a more engaging experience through gamification. Instead of using a traditional tracker alone, the app pairs budgeting tools with a virtual pet whose mood and wellbeing reflect the user’s financial habits. As users stay on track with their budgets, they unlock accessories and customizations for their monster, making progress feel rewarding and visible.

## Features

- **Gamified budgeting**
  - Track your financial habits through a virtual pet that responds to your spending behavior.
  - Unlock accessories and looks by maintaining strong budgeting habits.

- **Monthly budget tracking**
  - Organize spending across categories like rent, utilities, entertainment, and more.
  - Review current and previous months to compare how your habits evolve over time.

- **Interactive spending insights**
  - Visualize financial data with bar charts and pie charts.
  - Quickly understand where money is going each month.

- **Receipt scanning**
  - Scan receipts to automatically capture and store budgeting data.
  - Reduce manual entry and make expense tracking faster.

- **Receipt history**
  - Browse saved receipts with vendor, date, and total amount visible at a glance.
  - Review purchase history without digging through individual entries.

- **Category breakdowns**
  - View detailed summaries for each spending category.
  - See total spent, budget amount, receipt history, and AI-generated tips based on spending behavior.

## Tech Stack

- **Flutter** for cross-platform mobile development
- **Firebase** for storing receipts, budgets, and unlock data
- **OCR / receipt scanning tools** for extracting receipt data
- **AI-assisted insights** for spending tips and category guidance

## How It Works

Monster Money Manager stores user data in Firebase, including receipts, monthly budget categories, and customization unlocks. The app uses this data to generate spending visualizations, compare financial activity across months, and connect user progress to the virtual pet system. Receipt scanning helps users upload expense data with minimal effort, while category views provide more detailed insights into spending habits.

## Getting Started

### Prerequisites

Before running the project, make sure you have:

- Flutter installed
- A configured Firebase project
- A device emulator or physical device for testing
- Any required receipt scanning / OCR dependencies configured in the app environment

### Clone the repository

```bash
git clone https://github.com/MattWax04/monster_money_manager.git
````

### Install dependencies

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```

## Usage

1. Set up your monthly budget categories.
2. Scan or upload receipts to log purchases.
3. Review your spending through charts and category summaries.
4. Monitor your monster’s wellbeing as a reflection of your budgeting habits.
5. Unlock new accessories and appearances as you make progress.
